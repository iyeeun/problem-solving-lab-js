# Problem Solving Lab - JS

JavaScript의 기본 원리부터 알고리즘, 실전 문제 해결까지의 과정을 체계적으로 정리하고 기록합니다.

## Repository Structure

이 저장소는 다음과 같이 구성되어 있습니다.

### [00-js-core](00-js-core)

JavaScript 언어의 핵심 동작 원리와 주요 개념을 체계적으로 학습하고 정리합니다.

- books: 책을 통해 학습한 자바스크립트의 핵심 원리를 정리
- summary: 주요 자료구조 및 언어적 기능에 대한 요약

### [01-algo-core](01-algo-core)

알고리즘 문제 해결에 필수적인 데이터 구조와 알고리즘 이론을 정리합니다.

- 데이터 구조, 알고리즘 유형별 핵심 포인트 정리

### [02-problems](02-problems)

플랫폼별 코딩 테스트 문제 풀이와 템플릿을 보관합니다.

- 플랫폼 별 문제 풀이 내용 기록

### [99-notes](99-notes)

학습 과정에서의 인사이트와 주기적인 기록을 관리합니다.

- debugging: 디버깅 패턴
- logs: 월별 학습 기록 및 일일 회고 작성
- patterns: 자주 활용되는 알고리즘 패턴 정리
- performance: 성능 최적화 관련 내용
- templates: 특정 상황에 대한 재사용 가능한 템플릿 모음
- tips: 효율적인 문제 해결을 위한 코딩 팁 정리

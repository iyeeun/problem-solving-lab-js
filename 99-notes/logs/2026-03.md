#### 🧠 배운 내용

- `arr.every(cb)` : 배열 전체에 대해 조건을 만족하는지 검사
- array `fill()` vs `from()` : 동일한 값 (primitives)로 채우기 vs 매 요소마다 함수 실행하여 배열 생성

#### ❗ 막힌 내용

- 배열 초기화
- 문자열 뒤집기 : `str.split('').reverse().join('');`
- 문자열 중간 삽입 : `str.slice(0, start) + toAdd + str.slice(end + 1);`

#### 🔁 실수 패턴

- reduce로 합 구할 때 문자열이면 초기값 `0` 넣어줘야 함
